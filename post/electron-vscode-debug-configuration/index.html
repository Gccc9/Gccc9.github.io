electron-vscode-debug-configuration

2019-06-29

之前在Windows上配置VSCode调试Electon环境，一直调试不了，后面找到解决方案了，遂记录之。

直接上配置图(launch.json)：

{
    "version": "0.2.0",
    "configurations": [
      {
        "name": "Debug Main Process",
        "type": "node",
        "request": "launch",
        "cwd": "${workspaceRoot}",
        "runtimeExecutable": "C:/Users/Gccc/AppData/Roaming/npm/electron.cmd",//win10系统  
      //"runtimeExecutable": "${workspaceRoot}/node_module/.bin/electron", //MAC系统
        "program": "${workspaceRoot}/main.js",
        "protocol": "legacy"
      }
    ]
}

因为我的项目下没有node_module目录，所以原本${workspaceRoot}/node_module/.bin/electron.cmd需要修改为自己的路径。 我的路径是C:/Users/Gccc/AppData/Roaming/npm/electron.cmd。所以如此配置，ok，完工~
